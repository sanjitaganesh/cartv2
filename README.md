# Simple Interactive Shopping Cart

A shopping cart built using HTML, CSS, and vanilla JavaScript with dynamic quantity controls and real-time total calculation.

## Features
- Increase/decrease item quantities
- Automatic total price updates
- Clean, simple UI styling
- Fully client-side (no frameworks)

## Tech Used
- HTML5
- CSS3
- JavaScript (ES6)

## What I Learned
- DOM manipulation with `textContent`
- Handling button events
- Managing application state using arrays
- Recalculating values dynamically instead of patching totals
